More Selectors

In addition to the selectors previously mentioned, CSS syntax also offers more complex patterns as selectors.

Descendant selector

div p { background-color: #aaddaa; }

This rule will apply to the <p> tag. In this case, it will apply the specified property (color) to <p> tag anywhere inside a <div>.

Descendent selectors are written by simply placing a space between the elements.

Child selector

div > p { background-color: #aaddaa; }

The child selector operates in a similar manner, except that it will target only <p> tags that are direct child of an element (in this case, the <div>).

Child selectors are indicated by the > operator between the elements.

Attribute selector

img[alt=logo]{ margin: 2px }

The attribute selector will search for and apply property values to all elements with a given attribute. In the example above.

Attribute selectors are written by placing the targeted attribute in [ ].

Psuedo class

a:visited{ color: #efefef; }

CSS syntax also provides 'psuedo class' selectors. These selectors will apply properties to elements in a particular state. For instance, in the code above, the color will be applied to all links that have a 'pseudo-class' of 'visited.'

Psuedo class selectors are written by appending the psuedo class to the element after a colon ( : ) .
