Assigning Values

CSS Syntax uses several different ways of assining values.

Keywords

You can assign property values using keywords:

div{ border: thin; }

Dedicated keywords are available for only particular elements. For example, "thin" will set the size of a border, but not for fonts.

Physical measurments

You can indicate size of elements using physical measurements, such as in(inches), pt (points), pc (picas).

body{ margin: 1in; }

Screen measurements

Likewise, you can assign size by screen measurements by specifying the px (pixels).

div{ width: 150px; }

Relative Measurements

A popular way to set the size is using relative measurements, such as percentages (%) and em's. Relative measurements allow for your style to adapt to different devices and screens.

.container{ border: 80%; }

Color codes

Color can be assigned by hex code (#ffffff), red-green-blue values (rgb(r,g,b,), and name-text ('green').

font{ color: #ffffff; }

Fonts

Fonts are assigned using a case syntax ('font-family') that indicates the font you want to use, a generic font ending, and letterform.

body{ font-family: "Times New Roman", Times, serif; }

Functional Notation

There are also functional values that can be used. For example, a background can be set to an image using

background: url(img_location)
.

body{ background-image: url(image_location); }
