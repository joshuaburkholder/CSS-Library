Style Rules

Style rules follow a simple format:

selector { name: value; }

You can have multiple name/value pairs for in each rule.

selector { name1: value; name2: value; }

Selectors can be HTML elements, an given class, or an assigned id.

  • HTML selectors will select all of a given HTML element on a page. For instance, the following styling rule will change the background color of all the divs on a page to the color value given: div { background-color: #c13c3c }
  • Class selectors (indicated by a preceding .) will find any element with that class: .redText { color: red; }
  • ID selectors (deonted with a #) will find an element with that particular id: #thisDiv { background-color: rgb(60, 179, 193); }

Sidenote:

You'll notice in the example listed above, three different values were used to assign color - hex code (#c13c3c), text-code ('red'), and rgb code (rgb(60, 179, 193)). Any of these can be used to set the color property.

In addition to color, you can see the available name/property values that can be used to set styling rules at:
